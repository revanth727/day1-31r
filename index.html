Full Stack

html

The standard markup language used to structure content on the web, defining elements like headings, paragraphs, links, and images


css

CSS: A style sheet language for describing the appearance of a webpage, including layout, colors, fonts, and spacing.


javascript

JavaScript: A programming language that adds interactivity and dynamic behavior to websites, such as animations and user interactions


sql

SQL: A language for querying and managing relational databases, allowing operations like selecting, inserting, updating, and deleting data.


mysql

MySQL: A popular open-source relational database management system that uses SQL to store and retrieve data.


python

Python: A versatile, high-level programming language used in web development, data analysis, machine learning, automation, and more.


java

Java: An object-oriented programming language known for its portability across platforms, often used in enterprise applications and Android development.


nodejs

Node.js: A JavaScript runtime environment that allows running JavaScript on the server side, enabling fast, scalable web applications.


reactjs

React.js: A JavaScript library for building user interfaces, focusing on reusable components for creating dynamic and responsive web apps.


redux

Redux: A state management library for JavaScript apps, often used with React, to manage the application's state in a predictable way.
